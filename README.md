## Language Inputs
- ibus libpinyin

put this code block into .profile to use ibus inputs
``` sh
export GTK_IM_MODULE=ibus
export QT4_IM_MODULE=ibus
export QT_IM_MODULE=ibus
export XMODIFIERS=@im=ibus
```
